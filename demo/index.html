The Five Skills

Everything you'd do, automated.

Discover, score, qualify, warm-line, apply. Each skill is one markdown file in ~/.claude/commands/ — Claude Code picks them up automatically.

/job-radar
Daily ATS scan

Scrapes target companies via Greenhouse, Lever, Ashby. Scores against your lane regex. Outputs one top pick + 3 backups.

/job-radar-linkedin
Free Premium replacement

Ingests LinkedIn job-alert emails from Gmail. Parses subject-line cards. Auto-captures warm-line connections.

/job-radar-firecrawl
Broad-web discovery

Paid variant. Firecrawl search across the live web. Same scoring brain as the free pipeline. 10-15 credits per run.

/warm-line
Pre-apply leverage

Tier 1-4 connection ranking. Ready-to-paste outreach drafts. Halts cold-applies when ATS dedup risk exists.

/apply
End-to-end submit

JD URL to confirmation. WebFetch JD, tailor resume + cover, render to PDF, browser-MCP submit to Greenhouse / Lever / Ashby.

Architecture

One engine, one config, one SQLite file.

Skills orchestrate. Engine scores. SQLite persists. Your profile.yml at ~/.job-radar/profile.yml is the single source of truth — edit one file and every skill respects the change.

Quickstart

Four commands to running.

Skills land in ~/.claude/commands/. Engine lands in ~/.job-radar/engine/. Profile lives at ~/.job-radar/profile.yml. Idempotent — re-running upgrades in place.

bash # 1. Clone git clone https://github.com/Dicoangelo/job-radar-production.git cd job-radar-production # 2. Install (sets up ~/.job-radar/, copies skills to ~/.claude/commands/) bash install.sh # 3. Pick a starter profile and edit it cp examples/profile.revops.yml ~/.job-radar/profile.yml $EDITOR ~/.job-radar/profile.yml # 4. Restart Claude Code, then in any project: /job-radar
